Invariant and equivariant layers with applications to GNN, PointNet and Transformers

author: Marc Lelarge, course: dataflowr

Invariant and equivariant functions

As shown in the module on GNN, invariant and equivariant functions are crucial for GNN. For example, the message passing GNN (MGNN) layer is defined by:

hiℓ+1=f(hiℓ,{{hjℓ}}j∼i), h^{\ell+1}_i = f(h^\ell_i , \{\{ h^\ell_j\}\}_{j\sim i}), hiℓ+1​=f(hiℓ​,{{hjℓ​}}j∼i​),

where i∼ji\sim ji∼j means that nodes iii and jjj are neighbors and the function fff should not depend on the order of the elements in the multiset {{hjℓ}}j∼i\{\{ h^\ell_j\}\}_{j\sim i}{{hjℓ​}}j∼i​. This layer is applied in parallel to all nodes (with the same function fff) producing a mapping from hℓ=(h1ℓ…,hnℓ){\bf h}^\ell = (h^\ell_1\dots, h^\ell_n)hℓ=(h1ℓ​…,hnℓ​) to F(hℓ)=hℓ+1F({\bf h}^\ell) = {\bf h}^{\ell+1}F(hℓ)=hℓ+1 with F:Rn→RnF:\mathbb{R}^n \to \mathbb{R}^nF:Rn→Rn where nnn is the number of nodes in the graph (and only real hidden states are considered for simplicity). It is easy to see that FFF is an equivariant function, i.e. permuting its input will permute its output.

Another example of invariant and equivariant functions is given by the attention layer Attention(Q,K,V)=Z\text{Attention}(Q,K,V) = ZAttention(Q,K,V)=Z defined for QQQ a tensor of row queries, KKK the keys and VVV the values, Q,K,V∈Rn×dQ,K,V\in \mathbb{R}^{n\times d}Q,K,V∈Rn×d by

Zj=∑i=1nsoftmaxi(QjKiT)Vi. Z_j = \sum_{i=1}^n \text{softmax}_i(Q_jK_i^T) V_i. Zj​=i=1∑n​softmaxi​(Qj​KiT​)Vi​.

The queries are obtained from a tensor X∈Rn×cX\in \mathbb{R}^{n\times c}X∈Rn×c by Q=XWQTQ= XW_Q^TQ=XWQT​ and the keys and values are obtained from a tensor X′∈Rn×c′X' \in \mathbb{R}^{n\times c'}X′∈Rn×c′ by K=X′WKTK = X' W_K^TK=X′WKT​ and V=X′WVTV = X' W_V^TV=X′WVT​. We see that when the queries are fixed, the attention layer is invariant in the pair (keys, values):

Zj=∑i=1nsoftmaxi(QjKσ(i)T)Vσ(i), Z_j = \sum_{i=1}^n \text{softmax}_{i}(Q_j K_{\sigma(i)}^T) V_{\sigma(i)}, Zj​=i=1∑n​softmaxi​(Qj​Kσ(i)T​)Vσ(i)​,

hence Attention(X,X′)\text{Attention}(X,X')Attention(X,X′) is invariant in X′X'X′. Similarly, when the pair (keys, values) is fixed, the attention layer is equivariant in the queries:

Zσ(j)=∑i=1nsoftmaxi(Qσ(j)KiT)Vi, Z_{\sigma(j)} = \sum_{i=1}^n \text{softmax}_{i}(Q_{\sigma(j)}K_{i}^T) V_{i}, Zσ(j)​=i=1∑n​softmaxi​(Qσ(j)​KiT​)Vi​,

hence Attention(X,X′)\text{Attention}(X,X')Attention(X,X′) is equivariant in XXX. If X′=XX'=XX′=X, we get the self-attention layer so that SelfAttention(X)=Attention(X,X)\text{SelfAttention}(X) = \text{Attention}(X,X)SelfAttention(X)=Attention(X,X) is equivariant in XXX.

In this post, we will characterize invariant and equivariant functions following the ideas given in the paper Deep Sets.

Representation of invariant and equivariant functions

We start with some definitions.

For a vector x=(x1,…,xn)∈Rn{\bf x} = (x_1,\dots, x_n)\in \mathbb{R}^nx=(x1​,…,xn​)∈Rn and a permutation σ∈Sn\sigma \in \mathcal{S}_nσ∈Sn​, we define

σ⋆x=(xσ−1(1),…,xσ−1(n)) \sigma \star {\bf x} = (x_{\sigma^{-1}(1)},\dots, x_{\sigma^{-1}(n)}) σ⋆x=(xσ−1(1)​,…,xσ−1(n)​)

Definitions:

  • A function f:Rn→Rf:\mathbb{R}^n\to \mathbb{R}f:Rn→R is invariant if for all x{\bf x}x and all σ∈Sn\sigma \in \mathcal{S}_nσ∈Sn​, we have f(σ⋆x)=f(x)f(\sigma \star {\bf x}) = f({\bf x})f(σ⋆x)=f(x).

  • A function f:Rn→Rnf:\mathbb{R}^n\to \mathbb{R}^nf:Rn→Rn is equivariant if for all x{\bf x}x and all σ∈Sn\sigma \in \mathcal{S}_nσ∈Sn​, we have f(σ⋆x)=σ⋆f(x)f(\sigma \star {\bf x}) = \sigma \star f({\bf x})f(σ⋆x)=σ⋆f(x).

We can now state our main result:

Theorem

  • invariant case: let f:[0,1]n→Rf:[0,1]^n \to \mathbb Rf:[0,1]n→R be a continuous function. fff is invariant if and only if there are continuous functions ϕ:[0,1]→Rn\phi: [0,1] \to \mathbb R^nϕ:[0,1]→Rn and ρ:Rn→R\rho: \mathbb R^n\to \mathbb Rρ:Rn→R such that

f(x)=ρ(∑i=1nϕ(xi)) f({\bf x}) = \rho\left( \sum_{i=1}^n \phi(x_i)\right) f(x)=ρ(i=1∑n​ϕ(xi​))

  • equivariant case: let f:[0,1]n→Rnf:[0,1]^n \to \mathbb R^nf:[0,1]n→Rn be a continuous function. fff is equivariant if and only if there are continuous functions ϕ:[0,1]→Rn\phi: [0,1] \to \mathbb R^nϕ:[0,1]→Rn and ρ:[0,1]×Rn→R\rho: [0,1]\times \mathbb R^n\to \mathbb Rρ:[0,1]×Rn→R such that

fj(x)=ρ(xj,∑i=1nϕ(xi)) f_j({\bf x}) = \rho\left( x_j, \sum_{i=1}^n \phi(x_i)\right) fj​(x)=ρ(xj​,i=1∑n​ϕ(xi​))

We give some remarks before providing the proof below. For the sake of simplicity, we consider here a fixed number of points nnn on the unit interval [0,1][0,1][0,1]. For results with a varying number of points, see On the Limitations of Representing Functions on Sets and for points in higher dimension [0,1]d[0,1]^d[0,1]d with d>1d>1d>1, see On Universal Equivariant Set Networks and Expressive Power of Invariant and Equivariant Graph Neural Networks.

Our proof will make the mapping ϕ\phiϕ explicit and it will not depend on the function fff. The mapping ϕ\phiϕ can be seen as an embedding of the points in [0,1][0,1][0,1] in a space of high-dimension. Indeed this embedding space has to be of dimension at least the number of points nnn in order to ensure universality. This is an important remark as in learning scenario, the size of the embedding is typically fixed and hence will limit the expressiveness of the algorithm.

Coming back to the GNN layer (1), our result on the invariant case tells us that we can always rewrite it as:

hiℓ+1=ρ(hiℓ,∑j∼iϕ(hjℓ)), h^{\ell+1}_i =\rho\left( h_i^{\ell}, \sum_{j\sim i} \phi(h^\ell_j)\right), hiℓ+1​=ρ(hiℓ​,j∼i∑​ϕ(hjℓ​)),

and the dimension of the embedding ϕ(h)\phi(h)ϕ(h) needs to be of the same order as the maximum degree in the graph. Note that (8) is not of the form of (7) as the sum inside the ρ\rhoρ function is taken only on neighbors. Indeed, we know that message passing GNN are not universal (see Expressive Power of Invariant and Equivariant Graph Neural Networks).

As a last remark, note that the original PointNet architecture fff is of the form fi(x)=ρ(xi)f_i({\bf x}) = \rho(x_i)fi​(x)=ρ(xi​) which is not universal equivariant. Indeed, it is impossible to approximate the equivariant function gi(x)=∑ixig_i({\bf x}) = \sum_i x_igi​(x)=∑i​xi​ as shown below (we denote e1=(1,0,…,0){\bf e}_1=(1,0,\dots,0)e1​=(1,0,…,0)):

∥f(0)−g(0)∥2=nρ(0)2∥f(e1)−g(e1)∥2=(ρ(1)−1)2+(n−1)(ρ(0)−1)2≥(n−1)(ρ(0)−1)2, \|f(0) - g(0)\|^2 = n \rho(0)^2\\ \|f({\bf e}_1) -g({\bf e}_1)\|^2 = (\rho(1)-1)^2 + (n-1)(\rho(0)-1)^2\geq (n-1)(\rho(0)-1)^2, ∥f(0)−g(0)∥2=nρ(0)2∥f(e1​)−g(e1​)∥2=(ρ(1)−1)2+(n−1)(ρ(0)−1)2≥(n−1)(ρ(0)−1)2,

and these quantities cannot be small together. Hence PointNet is not universal equivariant but as shown in On Universal Equivariant Set Networks, modifying PointNet by adding the term ∑i=1nϕ(xi) \sum_{i=1}^n \phi(x_i)∑i=1n​ϕ(xi​) inside the ρ\rhoρ function as in (7) makes it universal equivariant. We refer to Are Transformers universal approximators of sequence-to-sequence functions? for similar results about transformers based on self-attention.

Proof of the Theorem

We first show that the equivariant case is not more difficult than the invariant case. Assume that we proved the invariant case. Consider a permutation σ∈Sn\sigma\in \mathcal S_nσ∈Sn​ such that σ(1)=1\sigma(1)=1σ(1)=1 so that f(σ⋆x)=σ⋆f(x)f(\sigma \star {\bf x}) = \sigma \star f({\bf x})f(σ⋆x)=σ⋆f(x) gives for the first component:

f1(x1,xσ(2),…,xσ(n))=f1(x1,x2,…,xn). f_1(x_1,x_{\sigma(2)},\dots, x_{\sigma(n)}) = f_1(x_1,x_2,\dots, x_n). f1​(x1​,xσ(2)​,…,xσ(n)​)=f1​(x1​,x2​,…,xn​).

For any x1x_1x1​, the mapping (x2,…,xn)↦f1(x1,x2,…,xn)(x_2,\dots, x_n) \mapsto f_1(x_1, x_2,\dots, x_n)(x2​,…,xn​)↦f1​(x1​,x2​,…,xn​) is invariant. Hence by (6), we have

f1(x1,x2,…,xn)=ρ(x1,∑i≠1ϕ(xi)) f_1(x_1,x_2,\dots, x_n) = \rho\left(x_1, \sum_{i\neq 1}\phi(x_i) \right) f1​(x1​,x2​,…,xn​)=ρ⎝⎜⎛​x1​,i​=1∑​ϕ(xi​)⎠⎟⎞​

Now consider a permutation such that σ(1)=k,σ(k)=1\sigma(1)=k, \sigma(k)=1σ(1)=k,σ(k)=1 and σ(i)=i\sigma(i)=iσ(i)=i for i≠1,ki\neq 1,ki​=1,k, then we have

fk(x1,x2,…,xn)=f1(xk,x2…,x1,…xn), f_k(x_1,x_2,\dots, x_n) = f_1(x_k,x_2\dots, x_1,\dots x_n), fk​(x1​,x2​,…,xn​)=f1​(xk​,x2​…,x1​,…xn​),

hence fk(x1,x2,…,xn)=ρ(xk,∑i≠kϕ(xi))f_k(x_1,x_2,\dots, x_n)=\rho\left(x_k, \sum_{i\neq k}\phi(x_i) \right)fk​(x1​,x2​,…,xn​)=ρ(xk​,∑i​=k​ϕ(xi​)) and (7) follows.

Hence, we only need to prove (6) and follow the proof given in Deep Sets. We start with a crucial result stating that a set of nnn real points is characterized by the first nnn moments of its empirical measure. Let see what it means for n=2n=2n=2: we can recover the values of x1x_1x1​ and x2x_2x2​ from the quantities p1=x1+x2p_1=x_1+x_2p1​=x1​+x2​ and p2=x12+x22p_2=x_1^2+x_2^2p2​=x12​+x22​. To see that this is correct, note that

p12=x12+2x1x2+x22=p2+2x1x2, p_1^2 = x_1^2+2x_1x_2+x_2^2 = p_2+2x_1x_2, p12​=x12​+2x1​x2​+x22​=p2​+2x1​x2​,

so that x1x2=p12−p22x_1x_2 = \frac{p_1^2-p_2}{2}x1​x2​=2p12​−p2​​. As a result, we have

(x−x1)(x−x2)=x2−p1x+p12−p22, (x-x_1)(x-x_2) = x^2-p_1x+\frac{p_1^2-p_2}{2}, (x−x1​)(x−x2​)=x2−p1​x+2p12​−p2​​,

and clearly x1x_1x1​ and x2x_2x2​ can be recovered as the roots of this polynomial whose coefficients are functions of p1p_1p1​ and p2p_2p2​. The result below extends this argument for a general nnn:

Proposition

Let Φ:[0,1]≤n→Rn\Phi:[0,1]_{\leq}^n \to \mathbb{R}^{n}Φ:[0,1]≤n​→Rn, where [0,1]≤n={x∈[0,1]n, x1≤x2≤⋯≤xn}[0,1]_{\leq}^n = \{ {\bf x}\in [0,1]^n,\: x_1\leq x_2\leq \dots\leq x_n\}[0,1]≤n​={x∈[0,1]n,x1​≤x2​≤⋯≤xn​}, be defined by

Φ(x1,x2,…,xn)=(∑ix1,∑ixi2,…,∑ixin) \Phi(x_1,x_2,\dots, x_n) = \left( \sum_i x_1, \sum_i x_i^2,\dots, \sum_i x_i^n\right) Φ(x1​,x2​,…,xn​)=(i∑​x1​,i∑​xi2​,…,i∑​xin​)

Let yk→y∈Im(Φ){\bf y}_k \to {\bf y} \in\text{Im}(\Phi)yk​→y∈Im(Φ), we need to show that Φ−1(yk)→Φ−1(y)\Phi^{-1}({\bf y}_k) \to \Phi^{-1}({\bf y})Φ−1(yk​)→Φ−1(y). We are now ready to prove (6). Let ϕ:[0,1]→Rn\phi:[0,1] \to \mathbb R^nϕ:[0,1]→Rn be defined by ϕ(x)=(x,x2,…,xn)\phi(x) = (x,x^2,\dots, x^n)ϕ(x)=(x,x2,…,xn) and ρ=f∘Φ−1\rho = f\circ \Phi^{-1}ρ=f∘Φ−1.
