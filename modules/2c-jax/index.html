Autodiff and Backpropagation

Jacobian

Let f:Rn→Rm\mathbf{f}:\mathbb{R}^n\to \mathbb{R}^mf:Rn→Rm, we define its Jacobian as:

∂f∂x=Jf(x)=(∂f1∂x1…∂f1∂xn⋮⋮∂fm∂x1…∂fm∂xn)=(∂f∂x1,…,∂f∂xn)=(∇f1(x)T⋮∇fm(x)T)\begin{aligned} \frac{\partial \mathbf{f}}{\partial \mathbf{x}} = J_{\mathbf{f}}(\mathbf{x}) &= \left( \begin{array}{ccc} \frac{\partial f_1}{\partial x_1}&\dots& \frac{\partial f_1}{\partial x_n}\\ \vdots&&\vdots\\ \frac{\partial f_m}{\partial x_1}&\dots& \frac{\partial f_m}{\partial x_n} \end{array}\right)\\ &=\left( \frac{\partial \mathbf{f}}{\partial x_1},\dots, \frac{\partial \mathbf{f}}{\partial x_n}\right)\\ &=\left( \begin{array}{c} \nabla f_1(\mathbf{x})^T\\ \vdots\\ \nabla f_m(x)^T \end{array}\right) \end{aligned}∂x∂f​=Jf​(x)​=⎝⎜⎜⎛​∂x1​∂f1​​⋮∂x1​∂fm​​​……​∂xn​∂f1​​⋮∂xn​∂fm​​​⎠⎟⎟⎞​=(∂x1​∂f​,…,∂xn​∂f​)=⎝⎜⎜⎛​∇f1​(x)T⋮∇fm​(x)T​⎠⎟⎟⎞​​

Hence the Jacobian Jf(x)∈Rm×nJ_{\mathbf{f}}(\mathbf{x})\in \mathbb{R}^{m\times n}Jf​(x)∈Rm×n is a linear map from Rn\mathbb{R}^nRn to Rm\mathbb{R}^mRm such that for x,v∈Rn\mathbf{x},\mathbf{v} \in \mathbb{R}^nx,v∈Rn and h∈Rh\in \mathbb{R}h∈R:

f(x+hv)=f(x)+hJf(x)v+o(h).\begin{aligned} \mathbf{f}(\mathbf{x}+h\mathbf{v}) = \mathbf{f}(\mathbf{x}) + h J_{\mathbf{f}}(\mathbf{x})\mathbf{v} +o(h). \end{aligned}f(x+hv)=f(x)+hJf​(x)v+o(h).​

The term Jf(x)v∈RmJ_{\mathbf{f}}(\mathbf{x})\mathbf{v}\in \mathbb{R}^mJf​(x)v∈Rm is a Jacobian Vector Product (JVP), corresponding to the interpretation where the Jacobian is the linear map: Jf(x):Rn→RmJ_{\mathbf{f}}(\mathbf{x}):\mathbb{R}^n \to \mathbb{R}^mJf​(x):Rn→Rm, where Jf(x)(v)=Jf(x)vJ_{\mathbf{f}}(\mathbf{x})(\mathbf{v})=J_{\mathbf{f}}(\mathbf{x})\mathbf{v}Jf​(x)(v)=Jf​(x)v.

Chain composition

In machine learning, we are computing gradient of the loss function with respect to the parameters. In particular, if the parameters are high-dimensional, the loss is a real number. Hence, consider a real-valued function f:Rn→g1Rm→g2Rd→hR\mathbf{f}:\mathbb{R}^n\stackrel{\mathbf{g}_1}{\to}\mathbb{R}^m \stackrel{\mathbf{g}_2}{\to}\mathbb{R}^d\stackrel{h}{\to}\mathbb{R}f:Rn→g1​Rm→g2​Rd→hR, so that f(x)=h(g2(g1(x)))∈R\mathbf{f}(\mathbf{x}) = h(\mathbf{g}_2(\mathbf{g}_1(\mathbf{x})))\in \mathbb{R}f(x)=h(g2​(g1​(x)))∈R. We have

∇f(x)⏟n×1=Jg1(x)T⏟n×mJg2(g1(x))T⏟m×d∇h(g2(g1(x)))⏟d×1.\begin{aligned} \underbrace{\nabla\mathbf{f}(\mathbf{x})}_{n\times 1}=\underbrace{J_{\mathbf{g}_1}(\mathbf{x})^T}_{n\times m}\underbrace{J_{\mathbf{g}_2}(\mathbf{g}_1(\mathbf{x}))^T}_{m\times d}\underbrace{\nabla h(\mathbf{g}_2(\mathbf{g}_1(\mathbf{x})))}_{d\times 1}. \end{aligned}n×1∇f(x)​​=n×mJg1​​(x)T​​m×dJg2​​(g1​(x))T​​d×1∇h(g2​(g1​(x)))​​.​

To do this computation, if we start from the right so that we start with a matrix times a vector to obtain a vector (of size mmm) and we need to make another matrix times a vector, resulting in O(nm+md)O(nm+md)O(nm+md) operations. If we start from the left with the matrix-matrix multiplication, we get O(nmd+nd)O(nmd+nd)O(nmd+nd) operations. Hence we see that as soon as m≈dm\approx dm≈d, starting for the right is much more efficient. Note however that doing the computation from the right to the left requires keeping in memory the values of g1(x)∈Rm\mathbf{g}_1(\mathbf{x})\in\mathbb{R}^mg1​(x)∈Rm, and x∈Rn\mathbf{x}\in \mathbb{R}^nx∈Rn.

Backpropagation is an efficient algorithm computing the gradient "from the right to the left", i.e. backward. In particular, we will need to compute quantities of the form: Jf(x)Tu∈RnJ_{\mathbf{f}}(\mathbf{x})^T\mathbf{u} \in \mathbb{R}^nJf​(x)Tu∈Rn with u∈Rm\mathbf{u} \in\mathbb{R}^mu∈Rm which can be rewritten uTJf(x)\mathbf{u}^T J_{\mathbf{f}}(\mathbf{x})uTJf​(x) which is a Vector Jacobian Product (VJP), correponding to the interpretation where the Jacobian is the linear map: Jf(x):Rn→RmJ_{\mathbf{f}}(\mathbf{x}):\mathbb{R}^n \to \mathbb{R}^mJf​(x):Rn→Rm, composed with the linear map u:Rm→R\mathbf{u}:\mathbb{R}^m\to \mathbb{R}u:Rm→R so that uTJf(x)=u∘Jf(x)\mathbf{u}^TJ_{\mathbf{f}}(\mathbf{x}) = \mathbf{u} \circ J_{\mathbf{f}}(\mathbf{x})uTJf​(x)=u∘Jf​(x).

example: let f(x,W)=xW∈Rb\mathbf{f}(\mathbf{x}, W) = \mathbf{x} W\in \mathbb{R}^bf(x,W)=xW∈Rb where W∈Ra×bW\in \mathbb{R}^{a\times b}W∈Ra×b and x∈Ra\mathbf{x}\in \mathbb{R}^ax∈Ra. We clearly have

Jf(x)=WT. J_{\mathbf{f}}(\mathbf{x}) = W^T. Jf​(x)=WT.

Note that here, we are slightly abusing notations and considering the partial function x↦f(x,W)\mathbf{x}\mapsto \mathbf{f}(\mathbf{x}, W)x↦f(x,W). To see this, we can write fj=∑ixiWijf_j = \sum_{i}x_iW_{ij}fj​=∑i​xi​Wij​ so that

∂f∂xi=(Wi1…Wib)T \frac{\partial \mathbf{f}}{\partial x_i}= \left( W_{i1}\dots W_{ib}\right)^T ∂xi​∂f​=(Wi1​…Wib​)T

Then recall from definitions that

Jf(x)=(∂f∂x1,…,∂f∂xn)=WT. J_{\mathbf{f}}(\mathbf{x}) = \left( \frac{\partial \mathbf{f}}{\partial x_1},\dots, \frac{\partial \mathbf{f}}{\partial x_n}\right)=W^T. Jf​(x)=(∂x1​∂f​,…,∂xn​∂f​)=WT.

Now we clearly have

Jf(W)=x since, f(x,W+ΔW)=f(x,W)+xΔW. J_{\mathbf{f}}(W) = \mathbf{x} \text{ since, } \mathbf{f}(\mathbf{x}, W+\Delta W) = \mathbf{f}(\mathbf{x}, W) + \mathbf{x} \Delta W. Jf​(W)=x since, f(x,W+ΔW)=f(x,W)+xΔW.

Note that multiplying x\mathbf{x}x on the right is actually convenient when using broadcasting, i.e. we can take a batch of input vectors of shape bs×a\text{bs}\times abs×a without modifying the math above.

Implementation

In PyTorch, torch.autograd provides classes and functions implementing automatic differentiation of arbitrary scalar-valued functions. To create a custom autograd.Function, subclass this class and implement the forward() and backward() static methods. Here is an example:

class Exp(Function):
    @staticmethod
    def forward(ctx, i):
        result = i.exp()
        ctx.save_for_backward(result)
        return result
    @staticmethod
    def backward(ctx, grad_output):
        result, = ctx.saved_tensors
        return grad_output * result
# Use it by calling the apply method:
output = Exp.apply(input)

Backprop the functional way

Here we will implement in numpy a different approach mimicking the functional approach of JAX see The Autodiff Cookbook.

Each function will take 2 arguments: one being the input x and the other being the parameters w. For each function, we build 2 vjp functions taking as argument a gradient u\mathbf{u}u and corresponding to Jf(x)J_{\mathbf{f}}(\mathbf{x})Jf​(x) and Jf(w)J_{\mathbf{f}}(\mathbf{w})Jf​(w) so that these functions return Jf(x)TuJ_{\mathbf{f}}(\mathbf{x})^T \mathbf{u}Jf​(x)Tu and Jf(w)TuJ_{\mathbf{f}}(\mathbf{w})^T \mathbf{u}Jf​(w)Tu respectively. To summarize, for x∈Rn\mathbf{x} \in \mathbb{R}^nx∈Rn, w∈Rd\mathbf{w} \in \mathbb{R}^dw∈Rd, and, f(x,w)∈Rm\mathbf{f}(\mathbf{x},\mathbf{w}) \in \mathbb{R}^mf(x,w)∈Rm,

vjpx(u)=Jf(x)Tu, with Jf(x)∈Rm×n,u∈Rmvjpw(u)=Jf(w)Tu, with Jf(w)∈Rm×d,u∈Rm\begin{aligned} {\bf vjp}_\mathbf{x}(\mathbf{u}) &= J_{\mathbf{f}}(\mathbf{x})^T \mathbf{u}, \text{ with } J_{\mathbf{f}}(\mathbf{x})\in\mathbb{R}^{m\times n}, \mathbf{u}\in \mathbb{R}^m\\ {\bf vjp}_\mathbf{w}(\mathbf{u}) &= J_{\mathbf{f}}(\mathbf{w})^T \mathbf{u}, \text{ with } J_{\mathbf{f}}(\mathbf{w})\in\mathbb{R}^{m\times d}, \mathbf{u}\in \mathbb{R}^m \end{aligned}vjpx​(u)vjpw​(u)​=Jf​(x)Tu, with Jf​(x)∈Rm×n,u∈Rm=Jf​(w)Tu, with Jf​(w)∈Rm×d,u∈Rm​

Then backpropagation is simply done by first computing the gradient of the loss and then composing the vjp functions in the right order.
